# Soulm8 printable models
This folder contains materials for the printable Soulm8 models.
## Printing guide
### soulm8_trident_indicator_unit.stl
* Transparent PETG or PC
* 1.6mm walls (4 wall count if 0.4mm)
* 100% infill density
* No supports required
* May be printed slower and 1% to 3% more flow than preset. You may [check out the article by Rygar1432 on Printables for the exact guide](https://www.printables.com/model/15310-how-to-print-glass).
    * Some materials are not truly transparent.
### Rest of the models
* Base color of the printer, ABS, ASA or PETG.
* 1.6mm walls (4 wall count if 0.4mm) 
* 40% infill density
* No supports required
